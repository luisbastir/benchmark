# Benchmark

## Description
- Language: Javascript (*.js)
- Framework: Node JS & Express JS
- Template Engine: Pug (*.pug)
- Database: MySQL

## Scripts
Install dependencies with:
```sh
$ npm install
```

Run migration script with:
```sh
$ npm run migrate
```

Run rollback script with:
```sh
$ npm run rollback
```

Run server with:
```sh
$ npm start
```
